# Retail Marketing Analytics Project

## Project Overview

This project analyzes retail supply and sales data to identify trends, supplier performance, and item-type sales insights.
The dataset was cleaned, transformed, stored in SQL Server, and visualized in Power BI.


## Dataset

* Source: Retail supply dataset
* Rows: 30,000
* Key fields: Year, Month, Supplier, Item Type, Retail Sales, Retail Transfers, Warehouse Sales, Date


## Tools & Technologies

* Python (Pandas, NumPy)
* SQL Server
* Power BI
* VS Code
* GitHub


## Project Workflow

1. Data Cleaning using Python
2. Feature creation (Total Sales, Date formatting)
3. Data stored in SQL Server
4. SQL queries for analysis
5. Power BI dashboard creation


## Dashboard Highlights

* KPI Cards for Total Sales, Retail Sales, Warehouse Sales
* Supplier performance visualization
* Item type sales analysis
* Monthly sales trend


## Project Purpose

To demonstrate skills in:

* Data Cleaning
* SQL Queries
* Data Visualization
* Business Insights generation
